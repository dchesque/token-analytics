========================================
❓ PROBLEMAS COMUNS E SOLUÇÕES v2024.2.0:
========================================

"Python não encontrado":
→ Instale Python 3.8+ de python.org
→ Ou use Microsoft Store (mais fácil)

"Módulo não encontrado":
→ Execute: pip install -r requirements.txt
→ Ou use o instalador automático (.bat)

"Emojis não aparecem (Windows)":
→ NORMAL: Sistema adaptou automaticamente
→ Funcionalidades não são afetadas

"APIs lentas":
→ NORMAL: Rate limiting inteligente (4s entre requests)
→ Evita banimentos e garante estabilidade
→ Primeira execução sempre mais lenta (sem cache)

"market_chart retorna 401":
→ NORMAL no plano gratuito CoinGecko
→ Sistema usa fallback automático para OHLC
→ Você ainda obtém dados históricos!

"Dados sociais limitados":
→ Configure API key LunarCrush (opcional)
→ Ou use dados gratuitos do CoinGecko
→ Sistema sempre funciona!
